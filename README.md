# iasc-tp2018
Distributed Messages Queue for "Implementacion de arquitecturas de software concurrentes".
Technoligies:
    - Docker
    - ZeroMQ
    - NodeJS
    - BeeQueue

# Prerequisites

- Node.js > 10.14.2
- Redis

# Usage

First run Redis Server

Go to /queueManager/app

```npm install```
```node main.js```

URL Arena http://0.0.0.0:4567
